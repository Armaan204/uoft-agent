UofT Agent Connector Privacy Policy

Last updated: 2026-04-07

Summary

UofT Agent Connector is a Chrome extension that helps a user import their own ACORN academic history into UofT Agent.

  • It does not collect usernames or passwords.
  • It does not automate login.
  • It only runs on https://acorn.utoronto.ca/*.
  • It only imports data after the user has logged in and manually clicks the import button.

Data Processed

When the user clicks Import Academic History, the extension may process visible academic-history data such as course code, title, credits, mark, grade, raw course text, import code, timestamp, and source page URL.

Data Not Collected

  • ACORN usernames
  • ACORN passwords
  • Browsing data from unrelated websites

How Data Is Used

The imported data is used only to let the user view their own ACORN academic history inside UofT Agent.

Storage

The extension stores the most recent import code and recent import payload locally in Chrome storage. The backend may store imported academic-history data so the UofT Agent app can read it back using the same import code.
