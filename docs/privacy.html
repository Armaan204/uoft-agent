Privacy Policy

Your AI academic assistant for University of Toronto.

Last updated: April 2026

Summary

UofT Agent and the UofT Agent Connector extension are designed to help University of Toronto students access academic information with minimal data collection. The extension collects only ACORN academic history that is already visible in the user's browser and only after the user explicitly starts the import.

What the extension collects

  • ACORN academic history content such as course codes, grades, credits, and session or term information shown on the academic history page.
  • A user-provided session import code used to associate the import with the matching UofT Agent session.
  • A timestamp and source page URL needed to support the import flow.

How data is collected

Data collection is user-triggered. The extension does not run an automatic scrape and does not collect anything until the user opens ACORN, logs in normally, pastes their import code, and clicks the import button.

What is not collected

  • No passwords.
  • No personally identifying information beyond the academic history content the user explicitly chooses to import.
  • No tracking, analytics profiling, or browsing data from unrelated websites.

How data is used

Imported ACORN academic history is sent to the UofT Agent Railway backend using the session import code. The import code is used to associate the uploaded data with the correct app session. Data is stored temporarily and associated only with that import code so the app can read it back for the user.

App processing

UofT Agent uses the Anthropic API for natural language processing so the application can answer academic questions and explain course information in natural language. Deterministic grade calculations and supporting application logic are handled by the app itself.
